Installation

There are three ways to install lib.js and lib.css in your file or project.

Npm

npm i @atmosoftx/mylib

Yarn

yarn add @atmosoftx/mylib

HTML

<link rel="stylesheet" href="https://atmosoftx.github.io/lib/lib.css"></script>
<script src="https://atmosoftx.github.io/lib/lib.js"></script>
