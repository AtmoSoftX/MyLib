Snackbar Information

Mylib have a css and js of snackbar. This is very simple to use. This is fully customable. You can change the Background color, Body color, Off color, Off bgcolor, Off focuscolor.

Snackbar Example

Normal Snackbar

Snackbar Code

<button class="btn primary-btn" onclick="msg('Message open successfully!!!', 'OK');">

Snackbar Example

Default Multiple Snackbar

Snackbar Code

<button class="btn red-btn" onclick="msg('Message open successfully!!!', 'OK', 'msg-red');">Open Message</button>
<button class="btn primary-btn" onclick="msg('Message open successfully!!!', 'OK');">Open Message</button>
<button class="btn green-btn" onclick="msg('Message open successfully!!!', 'OK', 'msg-green');">Open Message</button>
<button class="btn yellow-btn" onclick="msg('Message open successfully!!!', 'OK', 'msg-yellow');">Open Message</button>

Snackbar Code In CSS

Change the button last params of msg fuction to class which you want and define in css file or style tag and if you want primary msg use normal snackbar. 

.red-msg{{ '{' }}
  --msg-bgcolor: 255,0,0;
  --msg-focus-color: 255,0,0;
{{ '}' }}
.green-msg{{ '{' }}
  --msg-bgcolor: 0,255,0;
  --msg-focus-color: 0,255,0;
{{ '}' }}
.yellow-msg{{ '{' }}
  --msg-bgcolor: 255,255,0;
  --msg-focus-color: 255,255,0;
{{ '}' }}
