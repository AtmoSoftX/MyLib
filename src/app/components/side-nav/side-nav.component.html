Side Navbar Information

Mylib have a css and js of collapsing of navbar side navbar etc. You can use side navbar easily, this is used to collapse for left side but navbar used to collapse for up side and navbar push anything to down while side navbar is used to covers anything.

Side Navbar Example

Side Navbar Code 

<button class="btn black-btn" sidenav="#sidenav">
<aside class="sidenav" id="sidenav">
  <div class="p-10">
    <div class="d-flex align-items-center">
      <span class="ff-sans">Side Nav</span>
      <span class="flex-auto"></span>
      <button class="close btn-close trans-primary-btn" sidenav="#sidenav"></button>
    </div>
    <div class="ff-sans mt-20">
      This is side this is used add multiples links in a page.
    </div>
  </div>
</aside>
<div class="bg-sidenav"></div>

If you want to use multiples side navbar in a page the upper code position is last, copy and paste this code on up of upper code. And change the button sidenav attribute to #sidenav2 and also change the aside tag id to sidenav2 like this and div class bg-sidenav will use only 1 time in the last of document. 

<button class="btn black-btn" sidenav="#sidenav"><button>
<aside class="sidenav" id="sidenav2">
  <div class="p-10">
    <div class="d-flex align-items-center">
      <span class="ff-sans">Side Nav</span>
      <span class="flex-auto"></span>
      <button class="close btn-close trans-primary-btn" sidenav="#sidenav"></button>
    </div>
    <div class="ff-sans mt-20">
      This is side this is used add multiples links in a page.
    </div>
  </div>
</aside>
