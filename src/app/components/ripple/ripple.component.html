Ripple Information

Mylib have a css and js of ripple it is used to show user that the something is clicked like button is clicked and this is mostly used for button.

Ripple Example

Default Ripple

Ripple Code 

<button class="btn primary-btn ripple">Ripple</button>

Custom Ripple

<button class="btn red-btn" ripple ripple-fade-delay="600" ripple-fade-time="250" ripple-remove-delay="850">Expand, then fade</button>
<button class="btn red-btn" ripple ripple-expand-style="cubic-bezier(.12,.69,.8,.32)">Little pause ripple</button>
<button class="btn red-btn" ripple ripple-fade-delay="250" ripple-expand-delay="250" ripple-remove-delay="1250">Delay ripple</button>
<button class="btn red-btn" ripple ripple-disabled>Disabled</button>
<button class="btn red-btn" ripple ripple-centered>Centered</button>
<button class="btn primary-btn unbounded" ripple>Unbounded</button>
<button class="btn primary-btn" ripple ripple-bg="rgba(255,0,0,.5)">Colored</button>
<button class="btn primary-btn" ripple ripple-fade-time="1500" ripple-expand-time="1000" ripple-remove-delay="1500">Slow</button>
