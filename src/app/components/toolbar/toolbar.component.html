Navbar Information

MyLib have a css and js of collapse. You can use Navbar, I make a collapse in Navbar, Navbar has many colors like grey, black, primary.

Primary Navbar Code 

<nav class="primary-nav">
  <span class="nav-span">
    <h1 class="primary-nav-h1">Navbar</h1>
    <span space></span>
    <button class="primary-nav-btn" collapse="#collapse">
      <span class="material-icons">menu</span>
    </button>
  </span>
  <div id="collapse" class="collapse block-child">
    <a class="primary-nav-a nav-primary">Home</a>
    <a class="primary-nav-a">About</a>
  </div>
</nav>
